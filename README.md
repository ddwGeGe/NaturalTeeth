# 存在BUG问题

1、大部分漏洞都是本地搭建环境进行测试，小部分是内网实战遇到，实战中可能存在误报或者异常，如有情况可反馈，有空再优化优化（太菜 不一定短时间内能立马修复）

2、有些漏洞如果卡住或者误报，可代理到Burp抓包进行尝试

3、有些漏洞是反序列化上传或者执行命令，代理到Burp上不一定能抓到包，师傅可自行反编译查看，后续有空在优化一下 ~~
